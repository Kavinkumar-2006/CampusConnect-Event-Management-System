# CampusConnect - College Event Management System

CampusConnect is a web-based platform designed to manage college events efficiently. 
Students can view and register for events, while administrators can create and manage events.

## Features
- View upcoming college events
- Student registration for events
- Admin dashboard to manage events
- QR code for event registration
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
